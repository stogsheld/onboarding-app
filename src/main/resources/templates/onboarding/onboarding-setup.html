Onboarding - Setting up your PC Development Environment

Congratulations on joining our team! We're excited to have you on board.

As a new member of the HMRC IT team, you'll need to set up your PC to work within our development environment. This process will ensure that you have the necessary tools and access to collaborate effectively with your colleagues.

Step 1: Install Required Software

Download and install the following software:

  • Git
  • Visual Studio Code
  • Docker
  • Kubernetes

Once the installation is complete, restart your computer.

Step 2: Configure Git
  1. Open Git Bash.
  2. Create a new Git repository by typing the following command: git init.
  3. Add the following remote repository URL: https://github.com/HMRevenueCustoms/hmrc-development-environment.git.
  4. Fetch and merge the latest changes from the remote repository: git fetch && git merge origin/master.
Step 3: Configure Visual Studio Code
  1. Open Visual Studio Code.
  2. Install the following extensions:
    • GitLens
    • Docker
    • Kubernetes
Step 4: Configure Docker
  1. Open Docker Desktop.
  2. Sign in to Docker Hub.
  3. Pull the following Docker image: docker pull hmrc/development-environment
Step 5: Configure Kubernetes
  1. Open the Kubernetes dashboard.
  2. Create a new cluster.
  3. Deploy the following Kubernetes application: kubectl apply -f hmrc-development-environment.yaml
Step 6: Test Your Environment
  1. Open a new terminal window.
  2. Run the following command: kubectl exec -it hmrc-development-environment -- bash
  3. Inside the container, type ls to list the contents of the current directory.

If you've completed all of these steps, you should now have a working HMRC development environment. If you encounter any problems, please reach out to your team lead or the IT help desk for assistance.
